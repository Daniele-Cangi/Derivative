# Derivative

Derivative is a next-generation CLI AI agent whose cognitive architecture is grounded in the fundamental structures of reality: physics, mathematics, causality, formal logic, and quantum mechanics.

It does not simply use libraries as tools but reasoning through them as permanent cognitive lenses installed at the substrate level.

## Architecture

- **Cognitive Lenses**: DoWhy, SymPy, NetworkX, pgmpy, Qiskit, Z3, SciPy/Pint.
- **Cognitive Substrate Loader**: Base of the engine loading all lenses simultaneously.
- **Reasoning Kernel**: Consolidates framings and resolves logical flows using multi-frame reasoning.
- **Adversarial Validation Loop**: Attaches validation to assumptions and bounds.
- **Semantic versioned memory**: Recalls what has changed, utilizing deltas across sessions.
- **Epistemic Audit log**: Preserves analytical decisions and tags along the full history.

## Example Usage

```bash
derivative "explain the architecture of a distributed system"
derivative "what are the failure modes of this code?" --file main.py
derivative "is this reasoning valid?" --reasoning "..."
derivative --audit
derivative --memory
derivative --lenses
```
